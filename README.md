# EV Architecture Comparison

## Overview

This project compares the architecture of three real-world electric vehicles:

- Tata Nexon EV
- Hyundai Ioniq 5
- Ola S1 Pro

The objective is to understand differences in battery systems, motors, power electronics, charging technologies, and vehicle design while building a professional EV engineering portfolio.

## Objectives

- Compare EV architectures
- Study battery technologies
- Analyze motor technologies
- Learn charging systems
- Understand power electronics
- Build practical EV engineering knowledge
